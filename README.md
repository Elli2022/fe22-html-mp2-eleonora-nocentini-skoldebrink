Responsive restaurant website originally created for FE22 HTML Mini Project 2, then rebuilt with Astro for cleaner structure, shared layouts, and a GitHub Pages-friendly output.

## Live Site

- Production: `https://elli2022.github.io/fe22-html-mp2-eleonora-nocentini-skoldebrink/`

## Stack

- Astro
- TypeScript
- Custom CSS
- GitHub Pages publishing from `docs/`

## Local Development

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the local dev server:
   ```sh
   npm run dev
   ```
3. Create the production build:
   ```sh
   npm run build
   ```

## Version History

- The pre-modernization version is preserved in the tag `pre-astro-modernization-2026-05-31`.
- The current `main` branch contains the modern Astro rebuild.

## Deployment

GitHub Pages is configured to publish from the `docs` directory on `main`. Running `npm run build` refreshes that output.
